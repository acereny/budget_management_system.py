
 My Personal Expense Tracker

This is my first Python project where I practiced file handling and basic data visualization. I created this application to track daily expenses and visualize them using a bar chart.

## What does it do?
* It saves your expenses to a text file.
* You can categorize your spending (Food, Housing, etc.).
* It checks your budget and gives a warning if you spend too much.
* It uses the **Turtle** library to draw a bar chart of your spending.

How to use it?
Just run the `budget_management_system.py` file.
